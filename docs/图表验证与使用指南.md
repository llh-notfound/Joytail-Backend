## 🔧 故障排除

### 常见问题
1. **中文显示乱码**
   - 检查字体设置
   - 确保系统安装了中文字体

2. **图片分辨率不够**
   - 设置 `dpi=300` 或更高
   - 使用 `bbox_inches='tight'` 去除空白

3. **颜色不协调**
   - 参考企业VI色彩规范
   - 使用在线配色工具调整

4. **Mermaid渲染失败**
   - 检查语法格式
   - 简化复杂的嵌套结构
